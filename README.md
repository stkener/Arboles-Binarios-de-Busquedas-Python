# 🌳 Binary Search Trees & Linked Lists in Python

Implementation of fundamental **Data Structures** in Python, including **Binary Search Trees (BST)** and **Singly Linked Lists**, developed as part of a university Data Structures course.

The project models a music library where artists are stored in a Binary Search Tree and each artist maintains a linked list containing their songs.

The main objective was to understand how classic data structures work by implementing them from scratch without relying on Python built-in collections.

---

# 📚 Overview

The system organizes information using two custom data structures:

- Binary Search Tree (Artists)
- Singly Linked Lists (Songs)

Each artist is represented as a node in the tree, while each node stores a linked list containing all songs associated with that artist.

The project also includes a testing script and CSV dataset provided by the course instructors.
